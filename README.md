# 🛍️ LuxeCart – Premium E-Commerce Frontend

A modern, responsive, and premium frontend E-Commerce web application built using **React.js** and **Tailwind CSS**.

Designed with a luxurious dark theme, smooth animations, reusable components, and a clean user experience, this project demonstrates modern frontend development practices suitable for production-ready applications.
